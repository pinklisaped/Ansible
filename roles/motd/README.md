Dynamic motd configurator
=========

The role changes the welcome message when you log in to the server via ssh.

Requirements
------------

Tested on:
  - Ubuntu 20
  - Ubuntu 22.04
  - Rocky 9

--------------

Role doesn`t have vars.

------------
